# 机器狗设备检测工具

## 🚀 快速开始

您已经获得了一套完整的设备检测工具，帮助您找到机器狗设备的名称。

### 立即使用

1. **运行检测工具**
   ```
   .\device_detector.bat
   ```

2. **按照提示操作**
   - 先记录当前设备列表
   - 插入机器狗设备
   - 重新检测对比找到新设备

### 工具文件说明

| 文件名 | 说明 | 推荐度 |
|--------|------|--------|
| `device_detector.bat` | 完整的中文交互式检测程序 | ⭐⭐⭐⭐⭐ |
| `quick_detect.ps1` | PowerShell快速检测脚本 | ⭐⭐⭐ |
